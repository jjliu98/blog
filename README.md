## 运行

编译环境: Debian 12

node 版本: v20

pnpm 版本: 9.0.6

```bash

# 启用 pnpm
npm install -g pnpm

# 安装依赖
pnpm install

# 更新依赖
pnpm run update

# 运行
pnpm run dev

# 编译打包
pnpm run build

# 预览
pnpm run preview

# 博客发布到 https://jjliu98.github.io
pnpm run depoly

# 博客同布到 https://jjliu98.github.io
pnpm run sync

# 源码同步到 https://github.com/jjliu98/blog
pnpm run open

```
